
# Postfix Calculator Desu

Simple stack based postfix calculator written in rust.

I was bored in class...

## Quickstart

```console
$ cargo r --release
```

### Supported operators

```
'+' => Addition
'*' => Multiplication
```

### Example:

```console
$ cargo r --release

calc-desu> *
calc-desu> 10
calc-desu> 2
calc-desu> 3
EOF (ctrl + d)
60
```
